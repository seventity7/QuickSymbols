# QuickSymbols

QuickSymbols is a small Dalamud plugin that adds a quick symbol picker for some text inputs like Chat, Guest Book and Party Finder.

## Features

- Adds a `♥` button near the native ChatLog area.
- Adds a separate `♥` button below the Party Finder **Recruitment Criteria > Comment** input when that window is open.
- Clicking a symbol inserts it into the matching active text input without sending the message.
- CTRL+Click favorites/unfavorites symbols.
- Favorite symbols appear in a Favorites section above the main symbol list.
- The popup stays open after selecting symbols.
- Includes a draggable ChatLog button-position edit mode saved in the plugin config.
- All UI follows the native in-game theme selected by the player

## Position editing

Open QuickSymbols with the ChatLog `♥` button, `/qs`, or `/quicksymbols`, then click **Change button position**. While the button says **Editing button position**, drag the ChatLog `♥` button to the desired location. Click **Editing button position** again to finish and save the position.
